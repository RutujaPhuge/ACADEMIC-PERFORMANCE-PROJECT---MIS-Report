# ACADEMIC-PERFORMANCE-PROJECT---MIS-Report

# Summary
  
  This MIS Reprt Analyzes student academic performance using scores, study habits, and demographic factors. 
  The dashboard helps in understanding performance patterns and supports academic review through structured
  reporting.

# Key Visuals & Insights covered

  1) Student count by Gender using pivot table.
  2) Score comparison based on Parents Education.
  3) Distribution of students across Groups.
  4) Impact of Weekly Study Hours on scores.
  5) Average Score comparison by Gender.
  6) Subject wise average scores by group.

# Tools
 
  Advance Excel, Pivot Tables, Visuals.
